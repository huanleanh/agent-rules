# HTML/CSS Guidelines

- **HTML**: Use semantic HTML5 elements (`<header>`, `<main>`, `<section>`).
- **Accessibility**: Include `aria` labels where necessary. Ensure proper contrast.
- **CSS**: Use Modern CSS features (Grid, Flexbox, Variables).
- **Naming**: Use BEM (Block Element Modifier) or TailwindCSS utility classes.
- **Responsive**: Mobile-first design. Use media queries or container queries.
- **Performance**: Minimize DOM nesting. Optimize images.
- **Tools**: PostCSS, Autoprefixer.
