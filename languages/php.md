# PHP Guidelines

- **Version**: Use PHP 8.1+ features (Constructor property promotion, Enums, Readonly properties).
- **Standards**: Follow PSR-1, PSR-2, and PSR-12.
- **Frameworks**: Laravel/Symfony conventions.
- **Types**: Use strict types (`declare(strict_types=1);`). Always type-hint parameters and return types.
- **Composer**: Use Composer for package management.
- **Security**: Use prepared statements for SQL. Sanitize user input.
- **Testing**: PHPUnit or Pest.
