# 🤖 AI Agent Rules & Language Guidelines

This repository is a comprehensive collection of rules and best practices for AI coding assistants (like Cursor, Claude Code, Antigravity) and 15+ programming languages.

## 📁 Repository Structure

- **`tools/`**: Configuration files for specific AI tools.
  - `cursor/.cursorrules`
  - `claude/CLAUDE.md`
  - `antigravity/instructions.md`
  - `windsurf/.windsurfrules`
- **`languages/`**: Detailed guidelines for 15+ languages (TypeScript, Python, Swift, Go, etc.).
- **`prompts/`**: Universal rules and global prompting strategies.
  - `global_rules.md`

## 🚀 How to Use

### For Cursor
Copy the contents of `tools/cursor/.cursorrules` to your project's `.cursorrules` file.

### For Claude Code
Copy `tools/claude/CLAUDE.md` to your project root or use it as a reference for your system prompt.

### For Antigravity
Refer to `tools/antigravity/instructions.md` for internal consistency and prompting.

### For Language-Specific Guidance
Point the AI to the relevant file in the `languages/` directory to ensure it follows the latest best practices for your stack.

## 📖 Integration & Prompting Guide
For a detailed guide on how to integrate this repository with your existing projects and how to use these rules effectively in your prompts, check the [Integration Guide](./INTEGRATION.md).
